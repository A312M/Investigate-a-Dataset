Introduction

Dataset Overview

  • Contains information from 100k medical appointment reports in Brazil.
  • Focuses on determining whether patients showed up for their scheduled appointments.
  • Provides insights into patient characteristics, scheduled and actual appointment dates, and health-related factors.

Dataset Fields

  • PatientId: Unique identifier for each patient
  • AppointmentID: Unique identifier for each appointment
  • Gender: Gender of the patient (Male or Female)
  • ScheduledDay: Date when the appointment was scheduled
  • AppointmentDay: Date of the actual appointment
  • Age: Age of the patient
  • Neighbourhood: Location of the hospital
  • Scholarship: Indicates whether the patient is enrolled in the Brazilian welfare program Bolsa Famlia (1 for enrolled, 0 for not enrolled)
  • Hipertension: Indicates if the patient has hypertension (1 for yes, 0 for no)
  • Diabetes: Indicates if the patient has diabetes (1 for yes, 0 for no)
  • Alcoholism: Indicates if the patient has alcoholism (1 for yes, 0 for no)
  • Handcap: Indicates if the patient has a handicap condition (1 for yes, 0 for no)
  • SMS_received: Indicates if an SMS reminder was received by the patient (1 for yes, 0 for no)
  • No-show: Indicates whether the patient showed up for the appointment ('No' for showed up, 'Yes' for did not show up)

Research Questions

  • Does age influence the likelihood of missing an appointment?
  • Do patients enrolled in the Bolsa Famlia welfare program (The scholarship) show up more or less frequently for appointments compared to those who are not enrolled?

import matplotlib.pyplot as plt

def plot_attendance_by_group(data, group_column, group_labels):
    """
        This function  plots a bar chart and prints the average 'No-show' rate for any group 
        in the DataFrame by specifying the group column and labels
        It takes in 3 arguments:
            - The dataset
            - The group column
            - The group Labels to put on the bar chart
    """
    #Calculating average 'No-show' rate
    avg_no_show = data.groupby(group_column)['No-show'].mean() * 100
    
    #Counting the number of patients who attended and missed appointments
    attended = data[data['No-show'] == 0].groupby(group_column)['No-show'].count()
    missed = data[data['No-show'] == 1].groupby(group_column)['No-show'].count()
    
    #Bar chart
    x = range(len(group_labels))
    
    plt.figure(figsize=(10, 6))
    plt.bar(x, attended, color='blue', label='Attended')
    plt.bar(x, missed, bottom=attended, color='red', label='Missed')
    
    plt.xlabel(group_column)
    plt.ylabel('Number of Patients')
    plt.title(f'Number of Patients by {group_column} and Attendance')
    plt.xticks(x, group_labels)
    plt.legend()
    plt.show()
    
    #Printing the average 'No-show' rate
    for label, rate in avg_no_show.items():
        if label == 0:
            label = "No"
            print(f"{label}: {round(rate, 2)}%")
        elif label == 1:
            label = "Yes"
            print(f"{label}: {round(rate, 2)}%")
        else:
            print(f"{label}: {round(rate, 2)}%")
